# TravelManager 3.0

Proyecto nuevo: agenda de viaje PWA local-first.

Fase 1:
- UI móvil-first para agenda y tabla de datos.
- IndexedDB local.
- Import/export JSON, backup/restaurar.
- Edición local.
- PWA offline.

Fase 2:
- Añadir sincronización con Supabase.
- No backend manual.

Estructura del proyecto:
- public/: HTML, manifest, iconos.
- src/: lógica JS modular.
- styles/: CSS responsivo.
- pwa/: service worker.
- data/: muestra JSON y datos iniciales.
